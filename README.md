# My-first-assignment-in-HTML
My first assignment in HTML. - Modern Web Application Development course in SMIT 
my first task. In this assignment, you will create your resume webpage using basic HTML tags. The goal is to design a simple online resume that shows your personal details, education, skills, and experience. You will practice using different HTML tags like headings, paragraphs, lists, links, and definition lists.

This exercise will help you understand:

How to set up and structure a resume in HTML
The use of headings, paragraphs, and lists for organizing content
How to use a definition list for presenting information clearly
